
## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш_пользователь/ваш_репозиторий.git
   cd ваш_репозиторий
2. Создайте виртуальное окружение:
python -m venv venv
3. Активируйте виртуальное окружение:
На Windows:
    venv\Scripts\activate
На macOS/Linux:
    source venv/bin/activate
4. Установите зависимости:
    pip install -r requirements.txt
5. Cоздайте файл в корневом каталоге API_TOKEN.py и добавтье строку YOUR_BOT_TOKEN = "Ваш_токен"
6. Запустите проект:
    python main.py

Файлы и функции:

Main.py - запускает бота, проверяет API токен

db_func.py - управляет созданием базы, обновлением и извлечением данных пользователей

quiz_func.py - запускает квиз для пользователя, создает клавиатуру, выводит вопросы

quiz_handlers.py - обрабатывает текст, нажатие кнопок, считает статистику и завершает квиз

quiz.py - содержит вопросы и правильные ответы

requirements.txt - содержит необходимые зависимости
